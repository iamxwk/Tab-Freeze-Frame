# Tab Freeze Frame

自动挂起闲置标签页并显示可视化快照，节省内存的同时不丢失工作进度。

## 功能特点

- **自动挂起** - 超过指定时间的闲置标签页自动挂起
- **可视化快照** - 每个挂起的标签页显示截图预览
- **即时恢复** - 点击挂起页面任意位置即可瞬间恢复原标签页
- **网址白名单** - 支持正则表达式，指定永不挂起的网站
- **隐私优先** - 所有数据存储在本地，不会上传到任何服务器
- **国际化** - 支持英文和简体中文

## 安装使用

1. 打开 `chrome://extensions`
2. 启用「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `dist` 文件夹
5. 点击扩展图标打开弹窗
6. 可在设置页面配置超时时间和白名单

## 从源码构建

```bash
npm install
npm run build
```

构建产物在 `dist` 目录。

## 配置说明

- **挂起超时** - 标签页闲置多久后自动挂起（1-120 分钟）
- **网址白名单** - 匹配这些正则表达式的网址永不被挂起（每行一个）

## 技术栈

- React 19 + TypeScript
- Vite
- TailwindCSS 4
- Chrome Extension Manifest V3
